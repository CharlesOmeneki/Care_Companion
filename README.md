# Care Companion

A production-ready full-stack caregiving platform that connects individuals and families with trusted caregivers through a secure, modern web application.

## 🌐 Live Website

https://findcarecompanion.com

## 🚀 Features

- Secure user registration and login
- Email verification and password reset
- Real-time live chat using Flask-SocketIO
- AI-powered chatbot
- User and administrator dashboards
- Caregiver search and matching
- Contact forms with email notifications
- Responsive design for desktop and mobile

## 🛠️ Built With

- Python
- Flask
- PostgreSQL
- SQLAlchemy
- Flask-SocketIO
- JavaScript
- HTML5
- CSS3
- Jinja2
- Gunicorn
- Nginx
- Ubuntu Linux
- DigitalOcean
- Git & GitHub
